# Mireya

## Description

Mireya - backend app template

## Install

To install dependencies:

```sh
bun install
```

## Run for dev

```sh
bun src/app_main/index.ts
```

## Tests

```sh
bun test tests/hono_rpc.test.ts
bun test tests/trpc.test.ts
```

## Production

```sh
bun build.ts
bun dist/index.js
```
